最后编辑于2026/06/28 by:nood

⚠警告: root后 你所作出的一些行为可能会导致手机系统损坏

介绍 ​

本文的内容大部分都来自酷安 临时root/免解root/解BL是因为小米17系列8e5芯片发现漏洞然后掀起的这场风波 在2026/03/08的事情 也叫做38解锁节

通用操作流程 ​

在手机上 需要打开开发者设置(在设置连点系统版本号就可以成功打开开发者设置了 从设置中搜索开发者设置就能找到了) 找到USB调试 将USB调试打开 如果有USB安装也要把USB安装打开 首先我们需要一根数据线 插到手机和电脑上 手机上这时会弹出来一个USB调试的权限申请 点击始终允许通过此计算机调试然后点击允许

临时root ​

连接成功后 我们有以下几个选择

方法1 ​

我们需要去下载platform-tools 这个是谷歌官方的工具 包含了fastboot和adb命令 下载好以后放在电脑上的一个位置(是解压过的 并且里面有fastboot.exe和adb.exe) 并把整个文件夹添加到系统环境变量里面 然后打开cmd

方法2 ​

下载搞机助手 在扩展功能那里找到Cmd命令行并打开

方法1和方法2打开cmd以后 需要先将手机进入fastboot 也可以在cmd输入:

bash
adb reboot bootloader

执行完成会显示一个OK的字样 进入fastboot后(一般都是背景黑色 有字的那种就是) 然后在cmd输入:

bash
fastboot oem set-gpu-preemption 0 androidboot.selinux=permissive

执行完成会显示一个OK的字样 然后我们进入系统 可以直接长按电源键重启 也可以在cmd输入:

bash
fastboot continue

方法3 ​

找一个自动脚本双击.exe即可 会直接给你下载KernelSU 并且直接越狱
